# small+

Version of Small with preisntalled tools (my main OS)

##### Full Install on Termux

```bash
git clone https://github.com/smallOS-cyberwarfare/smallPlus
cd smallPlus
pkg install proot-distro
./installInProotDistro.sh
```

##### Usage

- Login into small+
```bash
small+
```
